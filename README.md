# Data Analysis Projects

Добро пожаловать в мой репозиторий **Data Analysis Projects**!
В этом репозитории собраны мои проекты по анализу данных, машинному обучению и разработке аналитических сервисов.
Каждый проект демонстрирует практическое применение аналитических навыков: от обработки данных и EDA до построения моделей и визуализации результатов.

---

## Список проектов

| Проект                                 | Описание                                                                                                                                                                                             |
| -------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **A/B Testing & Product Analytics**    | Итоговый кейс по продуктовой аналитике и A/B-тестированию для лендинга школы английского языка. Анализ гипотез, расчет выборок, проверка статистических гипотез и проектирование архитектуры тестов. |
| **Bank Deposit Prediction (ML)**       | Модель машинного обучения для предсказания, оформит ли клиент банковский депозит. Включает EDA, предобработку данных и обучение классификационных моделей.                                           |
| **Block Thermal Power Plant DB**       | Проект по проектированию и анализу базы данных тепловой электростанции. ER-диаграммы, нормализация данных, сценарии использования.                                                                   |
| **Ghana GDP Time Series Analysis**     | Анализ временного ряда ВВП Ганы за 62 года. Исследование трендов, стационарности, построение моделей ARIMA и прогнозирование.                                                                        |
| **HH Resume Salary Analysis**          | Анализ и подготовка данных резюме с hh.ru для предсказания заработной платы соискателей. EDA, предобработка и обучение моделей ML.                                                                   |
| **Hotel Rating Prediction (EDA & ML)** | Проект по анализу отзывов и характеристик отелей для предсказания рейтинга. Включает разведывательный анализ данных, обработку категориальных признаков и построение моделей.                        |
| **House Price Prediction ML Service**  | Сервис предсказания стоимости домов для агентства недвижимости. Модель машинного обучения + API на Flask для интеграции с внешними приложениями.                                                     |
| **Marketing Analytics Pilot Project**  | Пилотный проект по оценке эффективности интернет-маркетинга автодилера. Объединение данных GA и CRM, расчёт ROMI, визуализация через Power BI.                                                       |
| **Museum DB Practice**                 | Практика проектирования базы данных для музея. ER-диаграммы, структура таблиц (30+), сценарии использования, коллаж посещения музея.                                                                 |

---

## Используемые технологии

* **Python**, **pandas**, **numpy**, **scikit-learn**, **sklearn Pipeline**, **pickle**, **Flask**, **Jupyter Notebook**
* **Excel**, **Power BI**, **Drawio** (для работы с данными и визуализацией)
* **SQL** и проектирование баз данных (ER-диаграммы, нормализация)
