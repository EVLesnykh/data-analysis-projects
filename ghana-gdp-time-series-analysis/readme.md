# Анализ ВВП Ганы: исследование временного ряда и прогнозирование

## Описание проекта
Проект посвящён анализу временного ряда ВВП Ганы за 62 года. Цель — исследовать свойства ряда, построить модели ARMA/ARIMA и сделать прогноз на ближайшие годы. Результаты могут быть использованы для оценки экономической ситуации и поддержки принятия решений.

## Цели проекта
- Исследовать временной ряд: выявить тренд, сезонность и аномалии.
- Проверить стационарность ряда с помощью теста Дики-Фуллера.
- Построить и подобрать оптимальные параметры модели ARMA/ARIMA.
- Выполнить прогноз ВВП Ганы на три года и визуализировать результаты.
- Сделать выводы о поведении экономики страны.

## Этапы проекта
1. **Загрузка данных и первичный анализ**  
   Визуализация временного ряда, выделение тренда и сезонности, разделение данных на обучающую и тестовую выборки.
2. **Скользящее среднее (Moving Average)**  
   Сглаживание ряда и сравнение с исходным рядом.
3. **Проверка стационарности**  
   Тест Дики-Фуллера, дифференцирование ряда при необходимости.
4. **Построение моделей ARMA/ARIMA**  
   Подбор параметров p, d, q, сравнение моделей по AIC.
5. **Прогнозирование**  
   Прогноз для отложенной тестовой выборки, построение доверительного интервала.
6. **Выводы**  
   Анализ результатов, интерпретация прогнозов.

## Используемые инструменты
- Python: pandas, numpy, matplotlib, seaborn, statsmodels
- Jupyter Notebook
