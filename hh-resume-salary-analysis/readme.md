# Проект: Анализ резюме HeadHunter и подготовка данных для предсказания зарплаты  

## Описание проекта
Проект посвящён анализу и подготовке данных резюме с сайта **HeadHunter (hh.ru)**  
для последующего построения модели, которая автоматически определяет **примерный уровень заработной платы соискателя** на основе информации из резюме.

Основной фокус проекта — **качественная обработка данных и разведывательный анализ (EDA)**, необходимые для корректного обучения модели машинного обучения.

---

## Бизнес-задача
Компания HeadHunter планирует разработать модель, которая:
- оценивает ожидаемый уровень заработной платы кандидата;
- использует данные, указанные пользователем в резюме;
- работает корректно на очищенных и структурированных данных.

Задача проекта — подготовить данные таким образом, чтобы они были пригодны для построения ML-модели.

---

## Используемые данные

### Источники данных:
- **База резюме hh.ru**  
  Содержит информацию о кандидатах: пол, возраст, образование, опыт работы, график занятости, желаемая зарплата и др.

- **Данные о курсах валют**  
  Используются для приведения зарплат к единой валюте.

- Резюме:

## Используемые инструменты
- **Python**
- **Pandas** — обработка и анализ данных
- **NumPy** — численные расчёты
- **Matplotlib / Seaborn** — визуализация данных
- **Jupyter Notebook** — исследовательский анализ

---

## Этапы работы над проектом

### 1. Базовый анализ структуры данных
- изучение структуры таблиц;
- анализ типов данных;
- выявление пропусков и аномалий.

### 2. Преобразование данных
- обработка признаков заработной платы;
- приведение валют к единому формату;
- преобразование категориальных признаков;
- создание новых аналитических признаков.

### 3. Разведывательный анализ данных (EDA)
- анализ распределения заработных плат;
- исследование зависимости зарплаты от:
  - опыта работы;
  - уровня образования;
  - графика занятости;
  - региона;
- визуализация ключевых закономерностей.

### 4. Очистка данных
- обработка пропусков;
- удаление выбросов;
- корректировка некорректных значений;
- подготовка финального датасета для ML.

---

## 5. Метрика качества
Оценка проекта проводится по:
- корректности обработки данных;
- логике преобразований;
- качеству EDA;
- готовности данных к обучению модели машинного обучения.

## Освоенные навыки
- анализ и преобразование данных;
- разведывательный анализ данных (EDA);
- очистка и подготовка данных;
- визуализация данных в Python;
- подготовка датасета для ML-задач.

В результате проекта получен:
