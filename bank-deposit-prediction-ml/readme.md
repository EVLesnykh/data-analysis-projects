# Проект: Классификация клиентов банка для выявления склонных к открытию депозитов и повышения эффективности маркетинговой кампании

## Описание проекта
Цель проекта — определить характеристики клиентов банка, по которым можно прогнозировать склонность к открытию депозитов. Это позволяет повысить эффективность маркетинговых кампаний.

Проект включает анализ данных, обработку пропусков и выбросов, визуализацию данных, создание новых признаков, кодирование категориальных переменных и построение моделей машинного обучения.

## Бизнес-задача
Повысить результативность маркетинговой кампании банка за счет выявления клиентов, более склонных открыть депозит.

## Техническая задача
Построить модели классификации (логистическая регрессия, дерево решений) для предсказания целевой переменной `deposit` на основе характеристик клиентов.

## Данные
Данные о последней маркетинговой кампании банка включают:
- Клиентские характеристики: возраст, работа, семейное положение, образование, кредиты, баланс.
- Контакты: тип контакта, месяц и день последнего контакта, длительность контакта.
- История маркетинговых кампаний: количество предыдущих контактов и результат прошлой кампании.
- Целевая переменная: `deposit` — согласился ли клиент открыть депозит.

## Этапы работы
1. Загрузка и предварительный просмотр данных
2. Обработка пропусков и выбросов
3. Разведывательный анализ данных (EDA)
4. Преобразование данных: кодирование категориальных признаков, создание дамми-переменных, нормализация
5. Отбор признаков (SelectKBest)
6. Обучение моделей:
    - Логистическая регрессия
    - Дерево решений с подбором гиперпараметров (GridSearchCV)
7. Оценка качества моделей по метрикам классификации (accuracy, precision, recall, f1-score)
8. Визуализация важных признаков и дерева решений

## Результаты
- Наиболее значимые признаки: длительность контакта (`duration`), результат предыдущей кампании (`poutcome_success`), тип контакта (`contact_cellular`)
- Модели показали высокую точность на тестовой выборке, дерево решений позволило визуализировать ключевые факторы.
- Выводы помогут банку оптимизировать маркетинговые кампании и повысить вероятность открытия депозитов клиентами.
