# Marketing Analytics Pilot Project  

*(GA + CRM, ROMI, Attribution, Dashboard)*

## Описание проекта

Пилотный аналитический проект по оценке эффективности интернет-маркетинга для автодилера **«Премиум Авто»**.  
Цель проекта: показать, какие маркетинговые кампании приводят не только к заявкам, но и к реальным продажам.

Проект выполнен в формате консалтингового пилота на ограниченных данных по двум брендам: **BMW** и **Mercedes**.

---

## Бизнес-задача
Объединить данные из **Google Analytics** и **CRM**, восстановить пользовательские цепочки касаний  
и на их основе:

- оценить эффективность источников трафика;
- рассчитать ключевые маркетинговые показатели (ROMI, выручка, конверсии);
- подготовить аналитический дашборд для принятия управленческих решений.

---

## Используемые данные
В проекте используются данные из нескольких источников:

- данные веб-аналитики (Google Analytics);
- данные CRM-системы;
- справочник автомобилей;
- внешний источник — **курсы валют ЦБ РФ**.

## Используемые инструменты
Проект реализован с применением **трёх инструментов на разных этапах**:

- **Excel** — первичная обработка, расчёты, справочники  
- **Python** — объединение данных, расчёт показателей, прогнозирование  
- **Power BI** — визуализация и построение дашборда  

---

## Подготовка данных
В рамках подготовки данных были выполнены следующие шаги:

1. Загрузка исходных данных  
2. Выделение признаков марки и модели автомобиля  
3. Загрузка курсов валют с сайта ЦБ РФ  
4. Объединение всех источников в единую таблицу  
5. Расчёт недостающих показателей:
   - итоговая стоимость автомобиля в рублях  
   - маржа в рублях  
6. Формирование пользовательских цепочек касаний:
   - цепочки касаний  
   - сумма конверсий по цепочкам  
   - признак конверсии  
   - количество касаний для пользователя  

---

## Аналитика и расчёты
В проекте рассчитаны и проанализированы следующие показатели:

- регионы с наибольшим количеством заявок;
- средний процент отказов (Bounce Rate);
- распределение трафика по устройствам;
- наиболее конвертируемые источники;
- **ROMI** (с собственной методологией расчёта средней стоимости автомобиля) - показывает, насколько вложенные средства окупаются с точки зрения получаемой прибыли или стоимости продукта;
- выручка в рублях по долларовым позициям;
- наиболее выгодный источник трафика;
- анализ недостающих данных для расчёта чистой прибыли;
- прогноз количества конверсий до конца февраля;
- прогноз выручки за первый квартал при сохранении средней стоимости авто.

---

## Визуализация (Dashboard)
Подготовлен интерактивный дашборд со следующими элементами:

- цепочки касаний с цветовой индикацией продаж;
- карта продаж по регионам и городам;
- воронка продаж по моделям и маркам;
- продажи по источникам трафика с линией отсечки;
- декомпозиция продаж (Source / Medium / Campaign / Keyword);
- ежедневные продажи с визуализацией выполнения KPI;
- фильтры:
  - количество касаний  
  - признак конверсии  
  - бренд и модель  
  - регион  
  - источник трафика  
- среднее количество касаний в цепочке.
